# 作業提醒管理系統 (Homework Reminder Management System)

## 1. 專題概述
### 1.1 動機與背景
現在的學生課業繁忙，同時會有很多作業以及報告要繳交，為了避免繁忙中忽略或搞混截止日期而逾期繳交，本應用程式旨在管理時間並提醒作業截止日。

### 1.2 目標使用者
大學生，需要管理多門課程作業、報告、考試時間。

### 1.3 專題目標
- 提供作業、考試與報告的集中管理功能
- 透過提醒通知，降低忘記截止日期的情況
- 幫助使用者提早規劃時間，避免臨時抱佛腳
- 簡單好操作的頁面，提升使用便利性

## 2. 需求分析
### 2.1 功能性需求
- **F-01 截止日期追蹤與定時任務**：自動抓取或使用者設定作業截止日期，並觸發提醒。
- **F-02 即時同步繳交狀態**：勾選已完成後立即停止該項作業的後續提醒。
- **F-03 多方通訊軟體即時推送**：整合 LINE/Gmail 推送訊息。

### 2.2 非功能性需求
- **效能**：回應時間不超過三秒。
- **安全性**：登入帳號密碼驗證機制。
- **易用性**：介面簡單清楚，分類明確。

## 3. 技術規劃
- **前端**：HTML / CSS / JavaScript (純前端)
- **後端**：Python + Flask
- **資料庫**：SQLite
- **通知服務**：LINE Notify API
